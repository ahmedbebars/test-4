# ميثاق (Mithaq) - تطبيق الزواج الجاد

تطبيق أندرويد متكامل مصمم لتسهيل التعارف الجاد بغرض الزواج فقط، مع التركيز المطلق على الخصوصية، الأمان، والقيم المجتمعية العربية والإسلامية.

## 🚀 التحديث الأخير (Latest Enhancements)

لقد تم تطوير التطبيق ليصل إلى مستوى الاحترافية الكاملة (V1.0-Secure) من خلال الإضافات التالية:

*   **نظام التحقق الإلزامي (Mandatory Email Verification):** بوابة أمان جديدة تمنع دخول أي مستخدم إلا بعد تفعيل حسابه عبر رابط مرسل لبريده الإلكتروني الرسمي.
*   **دعم تسجيل الدخول بجوجل (Google Auth UI):** واجهات دخول عصرية تدعم خدمات جوجل والبريد الإلكتروني.
*   **تحسينات واجهة المستخدم (UI/UX Polish):**
    *   حركات انسيابية (Animations) عند فتح الملفات الشخصية.
    *   نظام "الإعجاب" (Like) بتفاعل حركي نابض.
    *   بطاقة "إكمال الملف" لتشجيع المستخدمين على زيادة جودة بياناتهم.
*   **ثبات البيانات (Cloud Persistence):** ربط كامل بـ Firebase Firestore لضمان حفظ الرسائل والبيانات للأبد حتى عند الخروج من التطبيق.
*   **نظام التنقل الذكي (Advanced Navigation):** إصلاح كامل لزر الرجوع (Back Button) في جميع الشاشات لتجربة تصفح سلسة.
*   **مركز الأمان (Safety Center):** دليل إرشادي كامل لضمان رحلة زواج آمنة وموثوقة.

## 🛠️ التقنيات المستخدمة (Tech Stack)

*   **اللغة:** Kotlin
*   **الواجهات:** Jetpack Compose (Modern Declarative UI)
*   **السحابة:** Firebase (Auth, Firestore, Messaging, Analytics)
*   **المعمارية:** MVVM (Model-View-ViewModel)
*   **التصميم:** Material Design 3
