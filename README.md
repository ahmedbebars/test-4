# ميثاق (Mithaq) - تطبيق الزواج الجاد

تطبيق أندرويد متكامل مصمم لتسهيل التعارف الجاد بغرض الزواج فقط، مع التركيز المطلق على الخصوصية، الأمان، والقيم المجتمعية العربية والإسلامية.

## 🚀 المميزات الرئيسية (Core Features)

*   **نظام توثيق متدرج:** شارات توثيق للصور والهوية الرسمية لضمان مصداقية المستخدمين.
*   **خصوصية ذكية (Privacy-First):** ميزة تمويه الصور تلقائياً (Blur) للحفاظ على الخصوصية حتى يتم القبول المتبادل.
*   **بحث وتوافق متقدم:** فلاتر دقيقة تشمل (الالتزام بالصلاة، الحالة الاجتماعية، الاستعداد للسفر، والموقف من التدخين).
*   **محادثات فورية آمنة:** نظام دردشة لحظي مرتبط بـ Firebase Cloud لضمان عدم ضياع الرسائل.
*   **مركز الأمان (Safety Center):** إرشادات توعوية متكاملة لضمان رحلة آمنة من التطبيق إلى اللقاء الواقعي.
*   **دعم الوضع الليلي واللغات:** واجهة عصرية تدعم الوضعين الفاتح والداكن، وباللغتين العربية والإنجليزية.

## 🛠️ التقنيات المستخدمة (Tech Stack)

*   **اللغة:** Kotlin
*   **الواجهات:** Jetpack Compose (Modern Declarative UI)
*   **السحابة:** Firebase (Authentication, Firestore Database, Analytics)
*   **المعمارية:** MVVM (Model-View-ViewModel)
*   **التصميم:** Material Design 3
